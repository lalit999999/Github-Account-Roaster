Design a **world-class, premium SaaS landing interface** for a product called **“GitHub Roast Tool 🔥”**. The product analyzes a GitHub profile and generates humorous AI-powered roast insights about a developer’s coding habits. The UI should feel **as polished as products from top tech startups** and must look visually impressive enough that users want to **screenshot and share their results on social media**.

OVERALL DESIGN DIRECTION
Create a **modern, clean, and high-end developer tool interface** inspired by premium SaaS products and modern developer platforms. The aesthetic should combine **minimalism, elegance, and playful humor**.

Visual inspiration:
• modern developer dashboards
• AI SaaS products
• sleek startup landing pages
• premium portfolio tools

The UI should look **polished enough to appear on Product Hunt or a startup landing page**.

GENERAL DESIGN CHARACTERISTICS
• minimal design with generous whitespace
• large modern typography
• soft gradients and glowing accents
• glassmorphism or translucent UI surfaces
• subtle animated UI elements
• layered depth using shadows and blur
• professional yet slightly playful tone
• highly shareable visual elements

The design must look **expensive, modern, and extremely clean**.

TECHNOLOGY CONTEXT
The design must be easy to implement using:

• Next.js
• Tailwind CSS
• TypeScript

Use UI patterns that translate well into Tailwind utility classes.

---

PAGE STRUCTURE

The page should contain the following sections:

1. HERO SECTION

Create a visually striking hero section centered on the page.

Background:
• subtle gradient background (dark purple to deep blue)
• soft glowing shapes or blurred gradient orbs in background
• faint grid or developer-style pattern overlay
• very subtle floating shapes for visual depth

Title:
Large headline:

“GitHub Roast Tool 🔥”

Style:
• bold modern developer font
• large size (48–64px on desktop)
• strong contrast
• slight gradient text effect

Subtitle:
“Let AI roast your coding habits.”

Style:
• medium weight text
• soft gray color
• max width for readability
• centered alignment

Spacing:
• generous vertical spacing between title, subtitle, and input

---

2. USERNAME INPUT COMPONENT

Place a **premium input component** below the hero text.

Container:
• centered card container
• glassmorphism background
• slight blur
• rounded corners
• subtle border
• soft shadow

Input field details:

• rounded corners (large radius)
• padding inside input
• GitHub icon placed inside the left side
• placeholder text:

"Enter GitHub username..."

• subtle inner shadow
• focus ring glow effect
• animated border highlight when active

Username preview feature:

When the user types a username:

• show GitHub avatar preview
• avatar appears inside input or next to button
• smooth fade-in animation

CTA button:

Label:
“Roast Me”

Button design:

• gradient background
• purple to blue gradient
• bold text
• rounded pill shape
• hover glow effect
• subtle scale animation on hover
• small flame icon next to text

Hover animation:
• button slightly lifts
• shadow increases
• gradient becomes brighter

---

3. LOADING STATE

When AI is generating the roast report:

Show a **loading card skeleton**.

Loading card design:

• same shape as roast card
• gray skeleton blocks
• animated shimmer effect

Loading text:

“Analyzing your coding crimes…”

or

“Scanning your repositories…”

Add playful small animation icons like:

• spinning gear
• code brackets
• small flame icon

The loading state should feel **fun and engaging**.

---

4. ROAST RESULT SECTION

After loading, display a **beautiful shareable roast report card**.

This is the **main visual highlight of the product**.

CARD DESIGN

Card container:

• rounded corners
• soft gradient background
• subtle noise texture
• glassmorphism effect
• soft drop shadow
• glowing border highlight

Card layout:

Top section:
• GitHub avatar (large circular)
• username
• roast score badge

Score badge:

Example:

“Roast Score: 82/100”

Badge style:

• gradient pill badge
• small flame icon
• bold text
• vibrant accent color

Middle section:

Roast bullet points (3–5).

Each roast line:

• bullet icon or flame icon
• humorous text
• balanced spacing

Example:

• You start projects faster than you finish them.
• Your README files appear to be optional.
• Half your repos haven’t been touched in years.

Typography:

• clear readable text
• slightly playful tone
• medium font size

Bottom section:

Category tag.

Example:

“Category: Weekend Hacker”

Style:

• small rounded badge
• muted background
• soft border

The roast card must look **perfect when shared as an image**.
